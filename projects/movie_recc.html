Genre-Based Movie Recommender System

A sophisticated movie recommendation system built with Python, Flask, and Flutter

Python Flask Flutter Machine Learning Azure Render

This project delivers a genre-based movie recommendation system, leveraging machine learning and cloud technologies to provide personalized movie suggestions.

Download APK

Project Overview

This system recommends movies based on genre similarity using cosine similarity metrics, with a Flask backend and Flutter mobile interface.

System Components

1. Data Processing Engine
  • Processes ml-25m dataset
  • One-hot encodes genres
  • Computes cosine similarity
2. Flask API
  • Azure cloud storage
  • RESTful endpoints
  • Deployed on Render
