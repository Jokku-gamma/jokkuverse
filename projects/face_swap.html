Project Description: Real-time Face Swap Application

This project showcases a real-time face swapping application that allows users to seamlessly replace faces in a live webcam feed or between uploaded images. Built with robust computer vision libraries, it demonstrates advanced image processing and interactive web development.

Technical Implementation:

  • Face Detection (Dlib): Utilizes Dlib's highly accurate HOG-based face detector to locate faces in both the input and target images/frames.
  • Facial Landmark Prediction (Dlib): Employs Dlib's pre-trained shape predictor to identify 68 key facial landmarks (e.g., eyes, nose, mouth, jawline). These landmarks are crucial for aligning and transforming faces.
  • Affine Transformation (OpenCV): Calculates an affine transformation matrix between the target face landmarks and the input face landmarks. This matrix is used to warp the target face onto the geometry of the input face.
  • Seamless Cloning (OpenCV): Leverages OpenCV's `seamlessClone` function (Poisson Blending) to smoothly blend the warped target face onto the input image. This technique minimizes visible seams and color differences, creating a natural-looking composite.
  • Gradio Web Interface: Provides a user-friendly web interface with Gradio, enabling real-time streaming from a webcam and easy image uploads for static face swaps.
  • Python Backend: All core logic for face detection, landmark prediction, and image manipulation is implemented in Python, ensuring efficient processing.
