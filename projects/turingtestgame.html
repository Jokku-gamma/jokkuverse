Turing Test Challenge

Can you tell the difference between human and AI? Play a game where you guess if an answer to a question comes from a human or an advanced language model.

GitHub Repository

Project Description: Handwritten Alphabet Recognition

The game challenges you to distinguish between human-written answers and those generated by advanced AI models (Gemini or OpenAI). Enter your API key, select a model, and see if you can beat the AI!

Technical Implementation:

  • Streamlit Frontend: The interactive user interface is built using Streamlit, providing a dynamic and responsive web application.
  • API Integration (Gemini & OpenAI): The game integrates with both Google Gemini and OpenAI APIs, allowing users to choose their preferred AI model for the Turing Test.
  • Human-crafted Responses: A curated set of human-written answers provides a challenging benchmark for comparison against AI-generated text.
  • Robust Error Handling: Includes comprehensive error handling for API calls and network issues, ensuring the game gracefully resets if a problem occurs.
  • Session State Management: Utilizes Streamlit's session state to manage game progress, scores, and user selections across interactions.
  • Customizable AI Persona: AI prompts are carefully crafted to make the AI's responses concise, practical, and human-like, avoiding common AI conversational patterns.
