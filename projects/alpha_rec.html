Project Overview

This project implements a complete deep learning solution for recognizing handwritten English alphabets. It combines a TensorFlow/Keras convolutional neural network (CNN) with a Gradio web interface to create an interactive application that allows users to draw letters and receive real-time predictions.

Technical Implementation

Deep Learning Model

The CNN model was trained on the EMNIST dataset, achieving 92% accuracy in classifying handwritten letters (A-Z).

Image Processing

User-drawn images are converted to grayscale, resized to 28x28 pixels, normalized, and reshaped for the model.

Key Features

Interactive Drawing Canvas

Draw letters with mouse or touchscreen, with adjustable brush size.

Real-time Feedback

Predictions update instantly as users draw.
