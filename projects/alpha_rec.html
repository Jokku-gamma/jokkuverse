Project Description: Handwritten Alphabet Recognition

This project implements a complete deep learning solution for recognizing handwritten English alphabets. It combines a TensorFlow/Keras convolutional neural network (CNN) with a Gradio web interface to create an interactive application that allows users to draw letters and receive real-time predictions.

Technical Implementation:

  • Deep Learning Model: The core of the application is a CNN trained on the EMNIST dataset, achieving high accuracy in classifying handwritten letters (A-Z). The model architecture includes convolutional layers for feature extraction and dense layers for classification.
  • Image Preprocessing: User-drawn images are converted to grayscale, resized to 28x28 pixels, and normalized to match the training data format. This preprocessing ensures compatibility with the trained model.
  • Real-time Prediction: The application processes user input in real-time, providing immediate feedback with the predicted letter and confidence scores.
  • Gradio Interface: The web interface is built using Gradio, a Python library for creating customizable UI components for machine learning models. The interface includes a sketchpad for drawing letters and displays the prediction results clearly.
  • Deployment: The application is containerized and deployed as a web service, accessible through any modern browser without requiring installation.

Key Features:

Interactive Drawing Canvas

Users can draw letters with their mouse or touchscreen, with adjustable brush size and clear functionality.

Real-time Feedback

Predictions update instantly as users draw, showing the top predicted letter with confidence percentage.

Mobile Responsive

The interface works seamlessly on both desktop and mobile devices.

Model Confidence

Displays the model's confidence level for the top prediction, providing transparency about the result.
