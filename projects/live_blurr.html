Project Description: Real-time Face and License Plate Blurring

This project provides a robust solution for privacy protection in video streams by automatically detecting and blurring sensitive information such as human faces and vehicle license plates. Utilizing state-of-the-art computer vision models, this application processes live webcam feeds or uploaded videos in real-time, making it ideal for surveillance, public safety, and content moderation.

Technical Implementation:

  • Face Detection: Employs a pre-trained Caffe model (MobileNet Single Shot MultiBox Detector) for efficient and accurate real-time face detection.
  • License Plate Detection: Utilizes an OpenCV Haar Cascade classifier (`haarcascade_russian_plate_number.xml`) for robust license plate identification.
  • Dynamic Blurring: Applies Gaussian blur with configurable intensity to detected regions, ensuring privacy while maintaining the overall context of the video.
  • Gradio Web Interface: Features an intuitive and interactive web interface built with Gradio, allowing users to enable/disable blurring for faces and plates, and adjust blur intensity on-the-fly.
  • Live Stream Processing: Optimized to process live webcam feeds seamlessly, providing a real-time demonstration of the blurring capabilities.
  • Deployment: Designed for easy deployment as a web service, accessible through a browser (as indicated by the `onrender.com` URL).

Key Features:

Real-time Privacy Protection

Automatically detects and blurs faces and license plates in live video streams.

Configurable Blurring

Users can toggle face and plate blurring on/off and adjust the blur intensity.

Webcam & Image Input

Supports live webcam input and allows uploading static images for processing.

Seamless Gradio Interface

Interactive and user-friendly web UI for controlling blurring parameters.
